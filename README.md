# Python Bank Management System

A Bank Management System developed using Python with a Streamlit-based web interface.

## Features
- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Account Details
- Update Account Information
- Delete Account
- Persistent data storage using JSON
- User-friendly web UI with Streamlit

## Technologies Used
- Python
- Streamlit
- JSON

## How to Run the Project
1. Clone the repository
2. Install dependencies:
   pip install streamlit
3. Run the app:
   streamlit run app.py
